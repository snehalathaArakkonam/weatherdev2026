# 🌦️ Weather App

A modern, responsive Weather App built using **HTML, CSS, and Vanilla JavaScript** that provides real-time weather updates and forecasts.

---

## 🚀 Live Demo

👉 [View Live Website](https://weatherdev2026.netlify.app/)

---

## 📌 Features

* 📍 Detects user’s current location using Geolocation API
* 🔍 Search weather by city name
* 🌡️ Displays real-time temperature and conditions
* 💧 Humidity, wind speed, feels like, min/max temperature
* ⏳ Hourly forecast (next 24 hours)
* 📅 5-day weather forecast
* 🌗 Light/Dark mode with localStorage
* 🎨 Modern UI with glassmorphism design
* 💡 Smart weather suggestions (e.g., carry umbrella, stay hydrated)
* 📱 Fully responsive (mobile, tablet, desktop)

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, Glassmorphism)
* JavaScript (ES6+)
* OpenWeatherMap API

---

## ⚙️ Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/snehalathaArakkonam/weatherdev2026.git
   ```

2. Open the project folder:

   ```bash
   cd weatherdev2026
   ```

3. Add your API key in `script.js`:

   ```javascript
   const API_KEY = "YOUR_API_KEY_HERE";
   ```

4. Open `index.html` in your browser
